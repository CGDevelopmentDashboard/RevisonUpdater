Revision Updater Helper

Getting Started

This powerful plugin automates the updating process, ensuring that revisions and their corresponding shared parameters are seamlessly integrated into your project.

Key features

  1. Revision Updater automatically updates revisions on sheets, saving you time and reducing the risk of errors.
  2. With Revision Updater, assigning newly created revisions to desired sheets is a breeze. The plugin provides users with the option to assign revisions to sheets as soon as they are created, streamlining the process and enhancing efficiency.
  3. Easily edit revisions and their corresponding shared parameters directly within title blocks. Revision Updater simplifies the editing process, ensuring accuracy and consistency across your project.
  4. From complex project browser hierarchies to conflicting sheet ownership, Revision Updater handles it all. No matter the complexity of your project, this plugin ensures smooth and reliable performance.
  5. Navigate sheets effortlessly using the same organizational structure defined in the Autodesk® Revit® Browser Organizations. Revision of Updatergroups sheets by Viewset, providing easy access to relevant information and improving project organization.

Functionality

  1. Automatically gives the user the option to assign the newly created revisions to the desired sheets as soon as created.
  2. Edits the revisions and their corresponding (rev.) shared parameters in the title blocks.
  3. Easy navigations for sheets the same way as it’s defined in the Revit Browser Organizations.
  4. Grouping sheets by Viewset
  5. Handling all cases of sheet project browser hierarchy
  6. Handling sheet ownership conflicting.
